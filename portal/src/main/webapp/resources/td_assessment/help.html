• When you launch an assessment, click the Start Assessment button to begin the test. Note that a timer will count down the available minutes on the top right of the page.
  • Read the instructions for each question carefully. Some have only one correct answer, while others have several correct answer.
  • You must answer each question before you can click the Next button to go to the next question.
  • If you want to review or change your answer, click the Previous button to go back to the previous question.
  • After you have answered all the questions, click the Submit Answers button on the last question page. The system will then calculate your score and show the result.
  • Click the Repeat Test button to retake the assessment. You cannot click this button once you have exhausted the maximum attempts allowed for the specific assessment.
  • To exit the assessment, ALWAYS use the Exit button to the top right (behind the Nobel Biocare logo).
